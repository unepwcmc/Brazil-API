Taxonify API

The following endpoints are available.

Search

Endpoint: /api/taxa?scientificName=&page=&perPage=

Params:

  • scientificName: string
  • page: number
  • perPage: number

Returns:

[ meta: [ count: number ], rows: [ { kingdomName: string, phylumName: string, className: string, orderName: string, familyName: string, genusName: string, scientificName: string, speciesId: number, speciesPlusId: number, gbifId: number, commonNames: string } ] ]

Clicking Breadcrumbs

For clicking the breadcrumbs there are two options. We can start by using the above endpoint, and use the breadcrumb string as the scientificName;

Species taxonomic data

Endpoint: /api/taxa/:speciesId

Params:

  • speciesId: number

Returns:

{ kingdomName: string, phylumName: string, className: string, orderName: string, familyName: string, genusName: string, scientificName: string, speciesId: number, speciesPlusId: number, gbifId: number, citesListings: array }

Species common names

Endpoint: /api/common_names/:speciesId

Params:

  • speciesId: number

Returns:

[ { id: number, name: string, language: string, regions: string } ]

Species distribution

Endpoint: /api/distribution/:speciesId

Params:

  • speciesId: number

Returns:

[ { id: number, region: string, } ]
